**Pfam sequences download**

_All sequences were downloaded on 9th of June 2025._


Pfams and their sequences play a large role in this study.
The Pfams were downloaded using perl scripts generated by Pfam itself.

**Step1:**
Go to Pfam, go to sequences and check the all box. Then check the download (fasta) box.
This will result in Pfam generating a script. Copy the script

**Step2.**
Run the perl script in bash.

_It is also possible to generate a python script, however sometimes Pfam generates a faulty script._


All Pfam sequences were downloaded and outputted in the .out of the bash script. Use bash commands to move it into the correct format

          mv slurm_output/perl03600_output%A.out > PF03600_sequences.fasta
